# 🔥 PromptForge Pro — Landing Page

The public marketing site for **PromptForge Pro**, the premium AI prompt library that 10x's your output.

## Deploy

This is a single static `index.html` built with Tailwind CDN — zero build step.

### Option 1: GitHub Pages (free, 60 seconds)
1. Repo Settings → Pages
2. Source: `Deploy from a branch` → `main` → `/` (root)
3. Site goes live at `https://dolarexxx.github.io/promptforge-landing/`

### Option 2: Vercel / Netlify
- Drag-and-drop `index.html`, done.

### Option 3: Custom domain
- Point `promptforge.pro` at GitHub Pages or Vercel — under 5 minutes.

## Connecting payments

Replace each `href="#"` on the pricing buttons with your Gumroad / Lemon Squeezy / Stripe checkout link.
